# 🎵 音频路由器 Audio Router

这个应用程序模拟VB Audio的功能，允许用户选择多个音频输入设备，将其混音并路由到多个输出设备。

![Audio Router](https://img.shields.io/badge/Python-3.11-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## ✨ 功能特点

- 🎤 **多输入支持**：选择多个音频输入设备（麦克风、虚拟设备等）
- 🔊 **多输出支持**：选择多个音频输出设备（扬声器、虚拟设备等）
- 🎛️ **实时混音**：自动混合多个输入源并路由到所有输出
- 🎨 **现代化UI**：美观的深色主题界面，设备分类清晰
- 📊 **设备检测**：自动检测并显示所有可用音频设备
- ⚡ **低延迟**：实时音频处理，延迟极低
- 🛡️ **错误处理**：完善的异常处理机制，稳定可靠

## 📖 使用方法

1. **启动应用**：运行 `run.bat` 或直接运行 `MultiIO.py`
2. **选择输入设备**：在左侧蓝色面板勾选一个或多个输入设备
3. **选择输出设备**：在右侧绿色面板勾选一个或多个输出设备
4. **开始路由**：点击 "▶ 开始路由" 按钮启动音频混音
5. **停止路由**：点击 "⏹ 停止" 按钮结束音频传输

💡 **提示**：
- 可以同时勾选多个输入和输出设备
- 支持滚动查看更多设备
- 应用会自动混合所有输入设备的音频

## 📦 依赖

- Python 3.11+
- pyaudio
- numpy
- tkinter (Python 内置)

## 🔧 安装依赖

```bash
pip install -r requirements.txt
```

⚠️ **注意**：
- pyaudio 需要安装 PortAudio 库
- Windows 用户可能需要安装 Visual C++ 运行库
- 确保有足够的系统权限访问音频设备

## 🎨 UI 特点

- **深色主题**：护眼的深色界面，适合长时间使用
- **设备分类**：输入设备（蓝色）和输出设备（绿色）分区明确
- **滚动视图**：支持大量设备的滚动浏览
- **状态指示**：实时显示路由状态和设备数量
- **响应式按钮**：清晰的开始/停止按钮，带状态变化

## 🐛 故障排除

### 问题：无法选择设备
- 确保设备已正确连接并被系统识别
- 检查设备驱动是否正常安装

### 问题：音频断断续续
- 尝试调整缓冲区大小（frames_per_buffer）
- 关闭其他占用音频设备的应用程序

### 问题：启动失败
- 确保已安装所有依赖
- 检查 Python 版本是否为 3.11+
- 查看终端错误信息
